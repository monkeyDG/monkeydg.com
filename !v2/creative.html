AVRA is an Ottawa-based 5-piece rock band, drawing inspiration from Led Zeppelin, U2, and The Black Keys.


We've played across Ottawa - from Petrie Island Beach on Canada Day, to RBC Bluesfest 2017, to an out-of-the-way chip truck event with 15 people that our guitarist promised would be our big break (...it wasn't).

Planetside Open Games is a matchmaking system for Planetside 2's dev-sanctioned competitive server. We've run over 2300 matches in the last year, and over 2,000 hours of match time.


Our 3-man dev team created an AWS-hosted discord bot written in Python with Mongodb for data processing and storage. The bot integrates with the game's API for scoring, Google Sheets for map reservations, and Teamspeak 3 for match notifications.
